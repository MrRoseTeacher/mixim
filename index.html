Purpose

Mixim is a learning game designed to make spelling and definition review interactive and engaging. Players must unscramble words using the provided description or definition as a hint.

Builder Page

You are currently visiting the builder page for the game, where you can specify your topic, word and description pairs, and a success message for the user when they finish. You can either fill out the form manually or use generative AI to create the content for you.

Fill out the form to generate a Mixim game link. The generated link will encode the information you've entered. Share the link with anyone you want to play the game. All fields must be completed in order to generate a game link, and there must be at least one word/description pair.

Select the 'Add Row' button to add more word/description pair. If there are more than one pair, you will also have the option to delete a word/description pair.

Using Generative AI

***Data centers, where generative AI is processed, use lots of electricity and fresh water. Try to use it sparingly and avoid reprompting if possible.***

You may use generative AI tools such as Chat GPT, Copilot, or Google Gemini to create game content for you. To do this:

  • Download the sample template file.
  • upload the same file in your chat window to help the AI give you the correct formatting.
  • Copy and paste the sample prompt into your chat. Make any necessary changes to the prompt.
  • Once the AI tool has given you a new file to use, you can upload it here to populate the form automatically.

SAMPLE PROMPT (Replace ## and words in CAPS)

Hello, I have a game shell created and I'd like your help generating content for the game. In the attached file, you can see the formatting. There needs to be a topic, pairs of words and descriptions/definitions, and a success message. It's important that the data file includes the information in that order. To be more specific with the words that are paired with the descriptions/definitions, please make sure that they are single words. Ideally, the words are less than 9 characters in length to avoid a line break on mobile phones.

Please generate a json file for me containing ## word/description pairs about TOPIC. INCLUDE ANY SPECIFIC NOTES.

The application uses UTF-8 characters, so if you need any symbols from other languages please use that formatting. If the content is math related, you may also use appropriate MathJax in the descriptions. Please note that the text will be imported into an input HTML tag, so please use escape characters in your MathJax code (such as \\neq instead of \neq).

Game Page

On the game page, users are greeted with a splash screen prompting them to start the game. Users can also select the 'remix' link in order to return to the builder page with the form pre-filled. If you like a game and would like to alter it, feel free to remix it and generate a new link.

Otherwise, if the user selects 'Start Game' they will see:

  • A progress meter in the top left.
  • An optional a timer in the top right.
  • Instructions and description as their hint.
  • A row of empty letter boxes.
  • A row of scrambled, but filled letter boxes.
  • A check answer button.
The user must drag the letter blocks into the empty ones and spell the word correctly. If the user is incorrect, the correct blocks will turn green and briefly hop. Incorrect blocks will turn red and briefly shake. The user must then drag letter blocks to swap letters and then check their work again. When the user get to the end, they will see the success screen.

Mixim is formatted for phones, tablets, and computers. The optimal experience is probably a tablet or a laptop with a touchscreen monitor. I hope you find Mixim useful!
