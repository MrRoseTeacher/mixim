Purpose

Mixim is a learning game that encourages interactive spelling and definition review practice. Users will have to unscramble words, using the description or definition as a hint.

Builder Page

This is the builder page for the game, where you can specify your topic, word and description pairs, and a success message for the user when they finish. You can use the form to create your game, or you can use generative AI to create the content for you. Please read the sample prompt section if you are interested in using AI.
